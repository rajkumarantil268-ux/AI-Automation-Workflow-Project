# AI-Automation-Workflow-Project:-
( https://us2.make.com/public/shared-scenario/DLebuQ5T32B/integration-perplexity-ai  )

AI-Powered LinkedIn Post Generator using Google Gemini and LinkedIn API.An AI-powered automation workflow that uses Google Gemini AI to generate professional LinkedIn content from user prompts and automatically publishes the generated content as LinkedIn posts.
🎯 Project Overview

This project demonstrates how Generative AI can be integrated with social media platforms to automate content creation and publishing. The workflow accepts a user prompt, sends it to Google Gemini AI for content generation, and automatically creates a LinkedIn post.

⚙️ Workflow Steps
Step 1: User Input

The user provides a topic or prompt.

Example:

Write a LinkedIn post about Data Analytics using Python.
Step 2: Google Gemini AI

The prompt is sent to Gemini AI.

Gemini:

Understands the request
Generates professional content
Creates LinkedIn-ready text
Step 3: Content Processing

The generated response is formatted for LinkedIn posting.

Features:

Professional tone
Hashtags
Call-to-action
Structured formatting
Step 4: LinkedIn Integration

The workflow connects to LinkedIn API and automatically publishes the generated content as a LinkedIn post.
